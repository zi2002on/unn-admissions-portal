# UNN Faculty Officer Admission Clearance Portal

A complete web-based dashboard for faculty officers to manage student admission clearance.

## Features
- Secure login with email + 5-digit PIN
- Real-time pie chart of admission status
- Clickable cards showing detailed student lists
- Approve/Decline with one click
- View uploaded files (O'Level, Birth Certificate, etc.)

## How to Run
1. Install Node.js
2. Open terminal in this folder
3. Run:
   ```bash
   npm install
   node server.js
